# EPIC-005: Saving And Sharing

Status: Backlog

## Goal

Let users keep, revisit, copy, and share recipes they have remixed.

## Why It Matters

The product becomes more valuable when good remixes are not disposable. Saving and sharing also create natural loops for return usage and feedback.

## Scope

- Included: local save or account-backed save, copy-friendly formatting, share links, and version naming.
- Not included: public profiles, comments, likes, feeds, or collaborative editing.

## Success Criteria

- A user can save a remix and return to it.
- A user can copy a clean cooking-friendly version.
- A user can share a remix link when sharing is enabled.
- Saved remixes preserve source context and remix settings.
